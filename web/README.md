# MyFarmShop Web App

MyFarmShop is a farm-to-client ecommerce experience built with Next.js. It
highlights a marketplace for fresh produce, farmer profiles, and an order flow
designed for reliable fulfillment.

## Highlights

- Marketplace catalog with featured products and live order snapshots
- Verified farmer network profiles
- Buyer and farmer workspaces with inventory, payments, and delivery tools
- CTA funnel for onboarding buyers and growers

## Database schema

The initial SQL schema is located at the repository root in `db.sql`.

## Getting started

```bash
cd web
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Build for production

```bash
cd web
npm run build
npm run start
```
